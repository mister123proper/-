# Анализ продаж и прибыльности компании
Данный проект выполнен в рамках формирования портфолио по бизнес-аналитике.

В качестве источника данных используется открытый датасет Superstore, загруженный с платформы Kaggle. Датасет моделирует деятельность розничной компании (супермаркета) и содержит информацию о заказах, клиентах, товарах, продажах, прибыли, скидках и регионах.

Цель проекта — провести комплексный анализ продаж компании на основе датасета Superstore (загружен с Kaggle), определить ключевые показатели эффективности бизнеса, выявить факторы, влияющие на прибыль, обнаружить убыточные направления и предложить рекомендации по повышению эффективности
В ходе проекта используются инструменты:SQL (PostgreSQL), Excel, Power BI

В рамках проекта планируется ответить на следующие бизнес-вопросы: 
- Какие категории товаров приносят наибольшую выручку?
- Какие категории наиболее прибыльны?
- Какие товары продаются в убыток?
- Как скидки влияют на прибыль?
- Какие регионы работают наиболее эффективно?
- Какие показатели можно использовать для оценки эффективности бизнеса?


Этапы проекта
- Проверка качества данных
- Исследовательский анализ данных
- Расчёт KPI
- Анализ продаж, прибыли и клиентской активности
- Построение аналитической отчётности в Excel
- Анализ данных с использованием SQL
- Создание интерактивного дашборда в Power BI

По итогам проекта будут:

- рассчитаны ключевые бизнес-показатели
- выявлены наиболее прибыльные и убыточные категории товаров
- проанализировано влияние скидок на прибыль
- подготовлен интерактивный дашборд
- сформулированы рекомендации для повышения эффективности бизнеса
