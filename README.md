# Анализ продаж и прибыльности Superstore в Excel

## Описание проекта

Проект выполнен в рамках подготовки портфолио бизнес-/BI-аналитика.

Цель проекта - провести анализ продаж компании Superstore, оценить ключевые бизнес-показатели, выявить прибыльные и проблемные направления, а также подготовить интерактивный Excel-дашборд для руководителя.

В проекте использовался датасет Superstore, содержащий данные о заказах, клиентах, товарах, категориях, регионах, продажах, скидках, количестве проданных товаров и прибыли.

## Инструменты

- Microsoft Excel
- Умные таблицы Excel
- Сводные таблицы
- Сводные диаграммы
- Срезы
- Формулы Excel
- Условное форматирование

## Этапы работы

### 1. Подготовка исходных данных

На первом этапе исходный датасет был приведён к формату умной таблицы Excel. Это позволило упростить дальнейшую работу с данными, использовать структурированные ссылки и корректно обновлять сводные таблицы.

### 2. Проверка качества данных

Была проведена базовая проверка качества данных:

- проверка пропусков;
- проверка дубликатов;
- проверка корректности дат;
- проверка отрицательных значений в продажах;
- проверка числовых полей;
- проверка диапазона дат.

Это позволило убедиться, что данные подходят для дальнейшего анализа.

### 3. Расчёт KPI

Были рассчитаны основные показатели эффективности бизнеса:

- общая выручка;
- общая прибыль;
- маржинальность;
- количество заказов;
- количество клиентов;
- количество проданных товаров;
- средний чек;
- средняя скидка.

Отдельно были рассчитаны KPI за 2014 год и проведено сравнение с 2013 годом.

## Сравнение 2014 года с 2013 годом

В проекте было проведено сравнение ключевых показателей 2014 года с предыдущим годом.

Основные изменения:

- выручка выросла примерно на 20,6%;
- прибыль выросла примерно на 14,4%;
- количество заказов выросло примерно на 29,2%;
- количество клиентов выросло примерно на 8,8%;
- средний чек снизился примерно на 6,6%;
- маржинальность снизилась примерно на 0,69 процентного пункта;
- количество проданных товаров выросло примерно на 27,5%.

Это показывает, что в 2014 году компания увеличила объём продаж и количество заказов, однако эффективность продаж немного снизилась: прибыль росла медленнее выручки, а средний чек стал ниже.

## 4. Анализ продаж

Была построена динамика продаж по месяцам.

Цель анализа - понять, как менялась выручка внутри 2014 года, были ли пики продаж и сезонные колебания.

Для этого использовались:

- сводная таблица;
- группировка дат по месяцам;
- линейная диаграмма продаж.

## 5. Анализ прибыли

Отдельно была построена динамика прибыли по месяцам.

Цель анализа - проверить, совпадают ли месяцы высокой выручки с месяцами высокой прибыли.

Это важно, потому что высокая выручка не всегда означает высокую прибыль.

## 6. Анализ категорий

Был проведён анализ по основным категориям товаров:

- Furniture;
- Office Supplies;
- Technology.

Для каждой категории были рассчитаны:

- выручка;
- прибыль;
- маржинальность;
- средняя скидка.

### Основной вывод по категориям

Категория Technology является наиболее прибыльной и показывает высокую маржинальность.  
Категория Furniture имеет заметно более низкую маржинальность, несмотря на значительный объём продаж. Это говорит о том, что внутри категории Furniture могут быть проблемные товары или подкатегории.

## 7. Анализ подкатегорий

Для более детального анализа был проведён разбор по подкатегориям.

Были определены:

- наиболее прибыльные подкатегории;
- наиболее убыточные подкатегории.

### Основные наблюдения

Наиболее прибыльные подкатегории:

- Copiers;
- Accessories;
- Phones;
- Paper;
- Appliances.

Наиболее проблемные подкатегории:

- Tables;
- Machines;
- Supplies;
- Bookcases.

Особенно выделяется подкатегория Tables, которая показывает отрицательную прибыль. Это означает, что товары данной подкатегории требуют дополнительного анализа: возможно, проблема связана с высокой скидкой, низкой маржой или себестоимостью.

## 8. Анализ регионов

Был проведён анализ прибыли по регионам.

Рассматривались регионы:

- West;
- East;
- South;
- Central.

### Основной вывод по регионам

Регион West показывает наибольшую прибыль и высокую маржинальность.  
Регионы Central и South имеют более низкую прибыльность, поэтому их стоит рассмотреть подробнее при дальнейшем анализе.

## 9. Интерактивный Dashboard

На отдельном листе был создан интерактивный Excel-дашборд для руководителя.

В дашборд вошли:

- KPI-карточки;
- динамика выручки по месяцам;
- динамика прибыли по месяцам;
- выручка по категориям;
- прибыль по категориям;
- убыточные подкатегории;
- прибыль по регионам.

Также были добавлены срезы для фильтрации данных:

- по году;
- по региону;
- по категории.

KPI на дашборде были связаны со сводными таблицами через функцию `ПОЛУЧИТЬ.ДАННЫЕ.СВОДНОЙ.ТАБЛИЦЫ`, благодаря чему показатели автоматически изменяются при выборе фильтров.

В результате был создан Excel-отчёт, который позволяет:

- быстро оценить состояние бизнеса;
- сравнить показатели 2014 года с предыдущим годом;
- анализировать продажи, прибыль, категории, подкатегории и регионы;
- находить проблемные направления;
- использовать интерактивные фильтры для анализа данных;
- демонстрировать владение Excel на уровне, необходимом для работы аналитиком.

Проект показывает навыки работы с данными, расчёта KPI, построения сводных таблиц, создания интерактивного дашборда.
